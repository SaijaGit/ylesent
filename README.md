## YleSent

### Clone project
```bash
git clone https://github.com/SaijaGit/ylesent.git
cd ylesent
```

### Poetry
```bash
poetry install
```

### Run all
```bash
poetry run ylesent
```

### Run specific parts
```bash
poetry run ylesent preprocess
poetry run ylesent train
poetry run ylesent predict
poetry run ylesent visualize
```
